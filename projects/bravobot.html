Overview

BravoBot is a personal AI assistant that listens to voice commands and performs useful tasks such as answering questions, automating browser actions, and controlling music. It combines speech recognition, NLP, and UI interaction into a single system designed for desktop use.

Key Features

The assistant supports:

  • Voice-to-text conversion using Whisper
  • Conversational responses powered by OpenAI APIs
  • Automation of actions like Google searches and opening apps using Selenium
  • Integration with YouTube Music for playback control
  • Simple graphical interface with real-time feedback

Technologies Used

Python, Whisper, OpenAI, Selenium, Tkinter, gTTS, Google APIs
