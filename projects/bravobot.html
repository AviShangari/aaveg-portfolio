Overview

BravoBot is a modular, voice-activated AI assistant that runs entirely from the command line. It transcribes user speech, classifies the intent using a fine-tuned transformer model, and performs appropriate actions using automation or generative AI. The system is designed to run offline-first with local LLMs like LLaMA via Ollama, but gracefully falls back to OpenAI when needed.

Motivation

Many desktop voice assistants either rely fully on cloud APIs or lack extensibility for personal use cases. BravoBot was created as a flexible alternative with an offline-first design, allowing users to control apps, manage notes, and ask AI questions without constant internet reliance. Because all voice input, intent classification, and even language model inference can be run locally, BravoBot ensures a high level of user privacy ‚Äî no data ever needs to leave your machine.

Key Features

  • Custom wake word detection ("Ben") for privacy and control
  • Intent classification using a fine-tuned DistilBERT model
  • Offline and online LLM support (Ollama and OpenAI GPT)
  • Voice-controlled commands: web search, open apps, tell time, check weather, play YouTube
  • Note-taking, note summarization, and note listing via CSV and LLM
  • Interruptible TTS using keyboard shortcut (Shift + Q)

Architecture & Workflow

Speech is captured using a Whisper model and converted to text. This text is passed through an intent classifier trained with DistilBERT to detect the user‚Äôs intent. Based on the prediction, automation handlers or LLM interfaces (local or cloud) respond accordingly. The assistant uses pyttsx3 for TTS and can be interrupted mid-sentence.

Technologies Used

Python, Whisper, DistilBERT, Hugging Face Transformers, PyTorch, Ollama, OpenAI GPT, pyttsx3, Selenium, Tkinter, REST APIs
