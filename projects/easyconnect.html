Overview

EasyConnect is a mobile-based automation tool designed during a hackathon to help beginners and elderly users control desktop functions with ease. It enables simple, tap-driven control of volume, brightness, searches, and more ‚Äî all from a smartphone, with no prior technical experience required.

Motivation

Many automation tools are built for technical users, requiring command-line usage or scripting. EasyConnect removes those barriers by focusing on intuitive UI, accessibility, and real-time responsiveness for everyday desktop tasks.

Key Features

  • Tap-based remote control interface from mobile to PC
  • Web search, volume adjustment, and brightness control
  • Built-in desktop automation using Selenium for launching apps and interacting with browsers
  • Designed with accessibility and ease-of-use as the top priority

Technologies Used

React Native, Flask, Python, Selenium (for desktop automation), REST APIs
